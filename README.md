# Lucky Todo List

![Demo](demo.gif)

## Description

Dockerized Todo List *by Lucky*

## Technologie

| | |
| ---- | ------- |
| API | Adonis JS |
| FRONT | React JS (vite) |
| DATABASE | MariaDB |
| SERVER | Apache2 |


## Access

| | | |
| ---- | ---- | ------- |
| Build | Front | localhost:80 |
| Dev | Front | localhost:8000 |
| Dev | API | localhost:3333/api/v1 |
|  | PhpMyAdmin | localhost:8888 |



## Installation

```bash
docker-compose up
```
